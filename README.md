# go-restapi-mysql
Building secure RESTful web API service using Golang, depedency manager (dep), routing (chi), and MySQL.

In generic CRUD application, API as follows:

1. POST /posts
2. GET /posts
3. GET /posts/{id}
4. PUT /posts/{id}
5. DELETE /posts/{id}
